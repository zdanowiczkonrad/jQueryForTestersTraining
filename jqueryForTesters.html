jQuery – lekka biblioteka programistyczna dla języka JavaScript, ułatwiająca korzystanie z JavaScriptu (w tym manipulację drzewem DOM). Kosztem niewielkiego spadku wydajności w stosunku do profesjonalnie napisanego kodu w niewspomaganym JavaScripcie pozwala osiągnąć interesujące efekty animacji, dodać dynamiczne zmiany strony, wykonać zapytania AJAX

suche fakty

  1. wydana w sierpniu 2006
  2. używana przez 55% z 10,000 największych stron
  3. najpopularniejsza biblioteka do JS na świecie
  4. używa jej ~20 mln. stron
  5. najlepsza i najgorsza rzecz jak przytrafiła się JS

krótko o jQuery

  • intuicyjne i łatwe API
  • zaawansowana manipulacja DOM-em
  • obsługa zdarzeń (event)
  • obsługa animacji
  • łatwy AJAX
  • wsparcie dla wszystkich przeglądarek
  • jQuery UI, jQuery Mobile
  • duże community
  • zajmuje 32 kB
